# Project3Unix Assembly

## Implemented:
* Looptest
* Pi
* Harmonic

## Extra Function:
* GCD

## Compile:
```
$ cd Project3Unix

$ make all

$ make clean all

$ ./x86-64-driver.sh desiredCalcFile.calc
```

## Run:
```
$ cd Project3Unix/bin

$ ./desiredCalcFile
```

## .s Files
The generated .s files are stored in the /lib directory
